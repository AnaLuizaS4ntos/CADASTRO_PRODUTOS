# Nebbia Inventory Control

Sistema de gerenciamento de inventário desenvolvido para a marca **Nebbia**, focado no controle de equipamentos audiovisuais e produtos de arte digital. 

Este projeto foi construído para demonstrar habilidades de desenvolvimento **Full-Stack**, unindo uma interface moderna com um backend funcional.

## Funcionalidades (CRUD Completo)
- **Create**: Cadastro de produtos com SKU, categoria, nome, cor, estoque e preço.
- **Read**: Visualização dos itens em uma tabela centralizada e estilizada.
- **Update**: Edição de informações de produtos já existentes.
- **Delete**: Remoção de itens do banco de dados com confirmação de segurança.

## Design & UX
O projeto utiliza conceitos de **Glassmorphism**, com painéis translúcidos, efeitos de desfoque (`backdrop-filter`) e uma paleta de cores baseada no verde limão da marca contra um fundo dark. 

## Tecnologias Utilizadas
- **Backend:** Python 3 + Flask
- **Banco de Dados:** SQLAlchemy (SQLite)
- **Frontend:** HTML5, CSS3 (Flexbox/Grid) e Jinja2
- **Versionamento:** Git & GitHub

## Como rodar o projeto
1. Clone o repositório:
   `git clone https://github.com/AnaLuizaS4ntos/seu-repositorio.git`
2. Instale as dependências:
   `pip install -r requirements.txt`
3. Execute a aplicação:
   `python app.py`
4. Acesse em seu navegador: `http://127.0.0.1:5000`
